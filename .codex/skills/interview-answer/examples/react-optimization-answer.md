# Example: React Optimization Answer

## Question

리렌더링 최적화 경험이 있나요?

---

## Answer

### 핵심 답변

네, 입력 시 전체 모달이 리렌더링되던 문제를 구조적으로 분리해 해결한 경험이 있습니다.

### 30초

입력 이벤트가 발생할 때마다 전체 모달이 갱신되는 구조였는데, 상태 구독 범위를 줄이고 필요한 컴포넌트만 렌더링되도록 분리해서 해결했습니다.

### 1분

리뷰 작성 모달에서 텍스트 입력 시 전체 컴포넌트가 리렌더링되는 문제가 있었습니다.  
이를 해결하기 위해 실시간 업데이트가 필요한 영역만 분리하고, 해당 컴포넌트에서만 상태를 구독하도록 설계했습니다.  
그 결과 입력 시 불필요한 렌더링이 줄어들고 사용자 입력 반응성이 개선되었습니다.
