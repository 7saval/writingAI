# Resume Bullet Template

다음 경험을 이력서 bullet 문장으로 변환하라.

## Input

- 작업 내용:
- 사용 기술:
- 문제 상황:
- 해결 방법:
- 결과:

## Requirements

- 한 줄 또는 두 줄로 압축
- “무엇 + 어떻게 + 결과” 구조 유지
- 기술 이름만 나열하지 말 것
- 성과 또는 변화가 드러나야 함

## Output Format

### Version 1 (기본)

문장

### Version 2 (강화)

더 임팩트 있게 다듬은 문장

### Version 3 (짧은 버전)

한 줄 요약
