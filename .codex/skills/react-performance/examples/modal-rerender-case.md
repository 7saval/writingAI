# Example: Modal Rerender Optimization

## 문제

입력 시 전체 모달 리렌더링 발생

## 원인

폼 상태를 상위에서 모두 구독

## 해결

- useWatch로 부분 구독
- 컴포넌트 분리

## 결과

렌더링 범위 축소, 입력 반응성 개선

---

## Output

### 문제 요약

입력 시 전체 모달이 불필요하게 리렌더링되는 문제

### 원인 분석

상위 컴포넌트에서 모든 상태를 구독하고 있었음

### 개선 방향

상태 구독 범위를 하위 컴포넌트로 분리

### 기대 효과

렌더링 감소 및 UX 개선

### 포트폴리오 문장

폼 상태 구독 범위를 분리해 입력 시 불필요한 리렌더링을 줄이고 사용자 입력 반응성을 개선했습니다.
